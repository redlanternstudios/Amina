# Amina

**Faith-centered AI companion app for Muslim women.**

Built by RedLantern Studios™

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Supabase (auth + database)
- Capacitor (iOS)
- RevenueCat (subscriptions)
- Groq / AI SDK (chat)

## Screens

## Getting Started

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env.local` and fill in your keys.
